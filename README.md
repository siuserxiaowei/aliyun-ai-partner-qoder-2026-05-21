## 内容结构

- `docs/index.html`：公开单页，包含产品定位、客户作战地图、案例库、合伙人行动清单和截图画廊。
- `docs/assets/slides/`：用户提供的 8 张现场截图素材。
- `sources/smart-summary.md`：从飞书智能纪要整理出的结构化来源摘要。

## 页面口径

- 纪要中部分产品名出现 `Codework`、`Codeway`、`CodeWar` 等转写差异；公开页统一按现场截图口径整理为 `Qoder`、`QoderWork`、`QoderWake`。
- 飞书内部临时图片链接、妙记跳转链接未放入公开页面。
- 页面定位是合伙人转介绍和客户演示用的一页式材料，不是逐字稿。
